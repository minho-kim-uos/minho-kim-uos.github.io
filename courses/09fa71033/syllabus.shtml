course objectives
  • to understand how the OpenGL graphics system works
  • to implement interactive 3D graphics applications
  • to get familiar with the newest graphics technology (shaders, OpenGL 3.0, etc.)
* We're NOT learning how to use 3D tools to build fancy objects!
