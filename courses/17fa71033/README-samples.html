How to build the course samples on Windows

(last updated: 5 Sep 2016)

test environment
  • platforms:
    • Windows 7 Professional K + Visual Studio 2010 Professional
    • Windows 10 Enterprise + Visual Studio Community 2015
  • freeglut: 3.0.0
  • glew: 2.0.0
  1. First, create a folder where all your OpenGL-related work will be stored. Let's call it my_OpenGL
  2. Create following directories.
    • my_OpenGL/include/GL (for header files)
    • my_OpenGL/lib/x32 (for 32bit LIB files)
    • my_OpenGL/lib/x64 (for 64bit LIB files)
    • my_OpenGL/bin/x32 (for 32bit DLL files)
    • my_OpenGL/bin/x64 (for 64bit DLL files)
  3. Set up for freeglut
    1. Download the pre-compiled libraries "freeglut 3.0.0 MSVC Package" at http://www.transmissionzero.co.uk/software/freeglut-devel
    2. Unpack it.
      * You can also find the link at the freeglut homepage http://freeglut.sourceforge.net
    3. Copy freeglut/include/GL/*.h to my_OpenGL/include/GL.
    4. Copy freeglut/lib/freeglut.lib to my_OpenGL/lib/x32.
    5. Copy freeglut/lib/x64/freeglut.lib to my_OpenGL/lib/x64.
    6. Copy freeglut/bin/freeglut.dll to my_OpenGL/bin/x32.
    7. Copy freeglut/bin/x64/freeglut.dll to my_OpenGL/bin/x64.
  4. Set up for glew
    1. Download the "Binaries" at http://glew.sourceforge.net
    2. Unpack it.
    3. Copy glew/include/GL/*.h to my_OpenGL/include/GL.
    4. Copy glew/lib/Release/Win32/glew32.lib and glew/lib/Release/Win32/glew32s.lib my_OpenGL/lib/x32.
    5. Copy glew/lib/Release/x64/glew32.lib and glew/lib/Release/x64/glew32s.lib to my_OpenGL/lib/x64.
    6. Copy glew/bin/Release/Win32/glew32.dll and my_OpenGL/bin/x32.
    7. Copy glew/bin/Release/x64/glew32.dll and my_OpenGL/bin/x64.
  5. Build a sample
    1. Create a project with
      • Template: Visual C++ → Empty Project
      • Name: my_proj (Whichever name you choose)
      • Location: my_OpenGL
      • "Create directory for solution" option unchecked
    2. Download triangles.cpp in my_OpenGL/my_proj and add it to the "Source Files" of the project just created.
    3. Select the "Solution Platform" (combobox on the left of the Run button)
      • 64bit: Select "x64"
      • 32bit: Select "x32"
    4. Set the project properties as follows:
      • C/C++ → General → Additional Include Directories: ../include
      • Linker → General → Additional Library Directories:
        • 32bit: ../lib/x32
        • 64bit: ../lib/x64
      • Linker → Input → Additional Dependencies: Add glew32.lib and freeglut.lib
      • Debugging → Environment
        • 64bit: PATH=%PATH%;../bin/x64
        • 32bit: PATH=%PATH%;../bin/x32
    5. Start build.
  6. Run the project from the Visual Studio.
