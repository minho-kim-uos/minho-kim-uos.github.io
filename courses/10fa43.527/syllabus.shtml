Computer Graphics for grads

fall 2010

Syllabus

  • Course info
    • Hours: 6-8 Mon
    • Instructor: Minho Kim ( Mr.Minho.Kim@gmail.com )
    • Office: 19-312 (tel: 2210-5786)
  • Course objectives
    • To understand interactive 3D graphics programming using OpenGL
    • To understand the spline theory and apply it in various research area
    • To understand GPGPU and apply it in various research area
    • To understand the latest research topics in computer graphics
  • Prerequisite
    • Basic knowledge of OpenGL programming
    • C/C++ or other language with OpenGL binding (see http://www.opengl.org/resources/bindings)
    • Data structure / algorithms
    • Linear algebra
    • ...fluency in English
  • Grading
    • 10% attendance
    • 20% presentation (1-2 talks for each)
    • 40% project (CG topics. Topics related to your own research area is recommended.)
    • 30% midterm exam (on spline theory)
    • Don’t expect a good grade if don’t try hard enough. This is not an easy-going course.
  • Lecture
    • Lectures will be given on theoretical topics. (e.g. spline theory)
    • Each student is required to pick 1~2 papers from cg conferences and give a talk.
    • Lectures will be given in Korean, but all the materials are in English.
  • Course materials
    • Resources on the web: Wikipedia, online documents, etc.
    • Refer to the course webpage of CG (for undergrads).
  • Topics (but not limited to...)
    • Spline theory
    • GPGPU
    • More topics in CG
    • Latest topics in CG conferences ( http://kesen.realtimerendering.com )
      SIGGRAPH, IEEE VIS, EG, PG, etc.
