GNU Octave 초간단 설명서

다운로드 및 설치

Octave-Force에서 윈도우즈용 설치 화일을 다운로드 받아 설치한다.

기초

  • GNU Octave 명령어 입력 창에서 여러 명령어를 입력하거나, 스크립트 화일을 만들어 한 번에 실행할 수 있다.
  • 명령어 끝에 ;를 붙이면 현재 명령어의 결과가 화면에 출력되지 않는다.

Linear Algebra

  • column vector 만들기: v=[1;2;3]
  • row vector 만들기: v=[1,2,3]
  • 2x3 크기의 matrix 만들기: m = [1,2,3;4,5,6]
  • transpose: v'
  • dot product: dot(u,v) 혹은 [1,2,3]*[1;2;3]
  • element-wise 곱셈: u.*v
  • rank: rank(m)
  • reduced row echelon form 만들기: rref(m)

현재 경로

  • 스크립트를 실행하려면, "현재 경로"를 스크립트 화일이 있는 위치로 변경해야 한다.
  • "현재 경로"를 알려면 명령어 pwd를 실행하면 된다.
  • 현재 경로를 바꾸려면 cd 명령어를 이용한다.
  • 디렉토리의 내용을 보려면 ls 명령어를 이용한다.

스크립트

  • Octave의 명령어 모임을 하나의 화일로 만든 후 실행할 수 있다.
  • 스크립트 화일의 이름을 "myfile.m"의 형태로 한다.
  • 스크립트를 실행하려면, 현재 경로를 스크립트 화일이 있는 경로로 변경한 후, (확장자 없이) 화일 이름을 명령어로 실행하면 된다. (ex: myfile)
