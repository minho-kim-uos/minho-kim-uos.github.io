Final Project

Implementing a Hack “Drawing” Assembly Program

( Due: 11:59pm 25 June 2018 (Monday) )

  • Goal: To implement a Hack assembly programs that draws a square dot on the screen.
    • Implement required assembly programs (Draw.asm) and send them to me (Mr.Minho.Kim@gmail.com).
  • Draw.asm
    • A program that draws a square dot in the screen.
    • The program should work correctly when executed in the following steps:
      1. Load the program Draw.asm into the CPU emulator.
      2. Clear all the memory.
      3. Execute the program.
      4. The grid is drawn and te square dot is drawn at the position (16,8).
      5. The user can move the dot around by pressing the arrow keys.
    • Requirements
      • You need to draw a grid on the screen. The size of each grid cell is 16x16. Therefore, overall 32x16 grid cells are drawn and the range of the (x,y) coordinates is [0,31]x[0,15].
      • The size of the square dot is 16x16.
      • You do not have to verify the x,y coordinates. You can assume they are in the valid range
      • When the dot is on the boundary, it should not move further.
    • Bonus points
      • Solve the “sensitive key” issue.
      • Draw a (filled) rectangular box.
      • Draw a (non-filled) rectangular box.
      • Resize the box using the keyboard.
  • Hints
    • Please read the CPU emulator tutorial first.
    • The “break points” feature might be really handy during debugging.
