Midterm Project: “Falling Stars”

  • Objective: To build an interactive animation “Falling Stars”
  • (Due: 30 Nov 11:59pm)
  • Requirements:
    • Render a terrain in the scene. The terrain can be a simple square, but the fancier it looks, the more points you can get.
      Suggested terrain rendering methods:
      • To render a fixed terrain using “height maps”.
      • To implement a “Procedural Terrain Generation” algorithm to generate the terrain in real time.
    • Implement a UI (mouse or keyboard) such that the user can interactively move the camera. (Refer to mouse_examine.cpp sample.)
    • Implement falling stars.
      • Each star should act as a point light source.
      • Make the intensity “attenuated” for more interesting effect.
      • Make the stars fall and the terrain is shaded accordingly. You can implement any fancier effect, e.g. fireworks.
      • Assign various colors to the stars so that the scene can be interesting.
      • Too many number of stars may slow down the rendering performance.
    • Be creative as much as possible!

      • which requirements you successfully implemented and
      • what you've done as extras, using which techniques, etc.
