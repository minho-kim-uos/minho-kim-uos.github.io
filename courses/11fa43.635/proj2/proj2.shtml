  • Objective: To implement a robot arm.
  • Executables
    • proj2.lin64: for linux 64 bit
    • proj2.xp32.exe: for Windows XP 32 bit
    Let me know if you have any problem with the executables!
  • Requirements
    • Render a container box with some objects (spheres, cubes, etc.) inside.
      • Render both "faces" and "edges" of objects to make them look 3D. (See sample05.c to see how to use glPolygonOffset.)
      • Use perspective projection for camera.
      • The aspect ratio should stay the same when the window is resized.
    • Implement an "interactive robot arm with claws".
      • By pressing the arrow keys, the arm should keep moving in the corresponding direction.
      • While moving, if the arrow key with the same direction is pressed again, the arm should stop moving.
      • While moving, if any other arrow key is pressed, the arm should keep moving in the new direction.
      • If the space bar is pressed, the arm should do the following action:
        1. opens the claw,
        2. stretches to the bottom,
        3. closes the claw to grab an object, and
        4. shrinks back.
      • The arm should grab an object if it lays close to the position of the arm. You can decide the "sensitivity" as you want. Just make it "look reasonable."
    • Make your program more interesting!!!
