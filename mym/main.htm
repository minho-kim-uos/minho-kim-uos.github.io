비누넷 웹메니저에 오신것을 환영합니다.

 * 제로보드 설치 
 최초 사용자의 경우 자신의 홈페이지 초기 인덱스 파일에서 제로보드 인스톨 바로가기 [GO] 메뉴를 클릭하여
 디비정보 입력 - 관리자정보 입력후 설치가 완료 됩니다. 
 설치 완료후 자동으로 제로보드 관리자페이지에 접속하게 되면, 최초 그룹 생성후 게시판관리/추가 메뉴를 
 이용하여 제로보드 생성/삭제등의 관리를 하실 수 있습니다.

 기존 사용자의 경우 제로보드를 재설치시 좌측 메뉴얼의 쉘메니저 안내글을 참조하여 텔넷접속후 메뉴선택으로
 제로보드 재설치를 쉽게 하실 수 있습니다.

 * MySQL DB ID : 신청하신 계정 아이디와 동일
 * MySQL DB Name : 신청하신 계정 아이디와 동일
 * MySQL DB Pass : 신청하신 계정 암호와 동일
 * Host Name : localhost
 * 각종 정보변경 신청 
 비누넷 - 설정정보 - 개인정보수정 게시판의 정보 수정 & 서비스 추가 양식글을 참조하여, 이용중 
 단위추가, 아이디/도메인/이메일변경 등의 각종신청을 하실 수 있습니다.
 신청방식은 계정신청서 작성방식과 동일하며 필요하신 부분만 복사하여 신청을 해주시면 됩니다.

 서비스 단위추가등의 유료서비스 신청시 신청서의 입금자명과 실입금자명이 다를경우 설정정보 - 입금정보
 확인에 양식을 참조하여 입금내역을 필히 안내해주셔야 처리가 가능합니다.
 * 단위추가 신청 
 계정이용중 계정용량이나 트래픽을 초과하셨을 경우 이용단위를 추가하여 서비스 허용수치를 추가하실 수
 있습니다. 단위추가는 남은달수에 대한 소요비용으로 계산이 되며 신청하실경우 아래와 같은 방법으로 안내를
 해드리고 있습니다.

 예) 단위추가 신청시 소요비용 안내 ----
 ### 등록일: 20031227 계정명: trueweb 계정종류: pro ###
 만기일: 20041231 신청일: 20040618 남은달수 : 6개월 13일(0.43개월)
 현재이용단위 : 1 단위 // 변경희망단위 : 2 단위
 남은달수 6.43개월 * 월소요비용 1000원 = 6430원을 입금하시면 됩니다
 입금후 비누넷-설정정보-입금정보확인 게시판에 입금내역을 안내하여 주시면 됩니다
 * 계정만기 안내 
 비누넷 계정 입주일 부터 1년후가 만기일이며, 만기 10일전에 안내 메일이 송신됩니다.
 계정 신청할때 작성하신 메일주소가 현재 유효한 메일 주소인지 확인을 하시고, 이상이 있다면 필히 정정 
 신청을 하셔야 합니다.
 자신의 입주일 과 메일주소 확인은 좌측 사용자정보 또는 텔넷 접속을 하셔서 쉘 메니져에서 확인이 가능 
 합니다. [ 비누넷 - 메뉴얼 참조 ] 

 예를 들어 입주일이 2002년 3월 8일 이라면 만기일은 2003월 10일이 됩니다.
 3월 01~10일 입주자 만기일 -> 3월 10일
 3월 11~20일 입주자 만기일 -> 3월 20일
 3월 21-말일 입주자 만기일 -> 3월 말일

 만기 메일 혹은 만기일이 임박한 회원분은 비누넷 - 설정정보 - 계정연장신청 메뉴의 양식글을 참조하여 
 연장신청서를 작성후 입금을 하시면 됩니다.
 * 도메인 구입 
 현재 비누넷에서는 도메인 등록대행 업무를 하고 있지 않습니다.
 http://dotname.co.kr 에서 저렴한 가격으로 도메인 구입을 하실 수 있습니다.
 도메인 구입시 네임서버 지정항목에서 비누넷 네임서버로 지정해주시면 됩니다.
 * 도메인 연결(포워딩)신청 
 1.도메인 구입 하신 곳에 가신후에, 네임서버 정보를 비누넷 네임서버 정보로 변경 합니다.
   1차 ) ns.byus.net (210.118.169.13)
   2차 ) ns2.byus.net (210.118.169.42)

 2.변경 신청을 한 시점부터 24시간이 지난후에, 비누넷에서 도메인 포워딩 신청을 합니다.
   비누넷 - 설정정보 - 개인정보 수정 메뉴에서 도메인 포워딩 신청 양식을 참조하여, 신청을 하시면 됩니다.
   계정신청시 도메인 정보를 기재하신경우 별도 포워딩 신청을 하실 필요는 없습니다.
   
 3.도메인 포워딩 셋팅후 기존주소 -> 신규주소로 일괄변경 서비스를 하고 있으며, 일부 swf 파일등은
   직접 수정해주셔야 합니다.

 주의 : 독립도메인 이용을 하시게 되면 id.byus.net 과 id at byus.net 은 회수가 되며 더이상 이용을 하실
        수 없습니다.
