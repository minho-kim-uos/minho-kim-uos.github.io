Midterm Project “Game of Life”

(due: 11:59pm, 4 May)

  • Objective: To implement Conway's Game of Life using OpenCL
  • Requirements
    • Download the stub script (GoL.py) and a sample seed file (seed.txt)
    • Usage:
      python GoL.py <seed_file> <num_iter>
      • <seed_file>: seed file name
      • <num_iter>: number of iterations
    • The seed file is a text file containing the seed pattern. You can assume that the rows and columns are always multiple of 8.
    • Currently, if you run the stub, it just
      1. loads the seed file into a numpy array (numpy.int32 type),
      2. displays the pattern,
      3. does NOT do any iteration, and
      4. saves the pattern as a text file named out.txt.
    • You should implement all the necessary OpenCL parts to do the iterations and save the final pattern as out.txt.
    • Extra points if “local memory + local synchronization” is used for better performance.
